# FlowMind Analytics — Brand Package

This package includes vector (SVG, PDF) and raster (PNG, ICO) assets for the FlowMind brand.

## Files
- **SVG**: icon, horizontal lockup, wordmark
- **PNG**: transparent background, multiple sizes
- **Favicon**: multi‑size ICO
- **Colors**: palette.json

## Colors
- Primary Blue: #0F62FE
- Accent Green:  #10B981
- Text Dark:     #111827
- Secondary Gray:#6B7280

## Usage
- Prefer SVG or PDF for print and high‑DPI.
- For app/web, use PNG sizes as needed. Keep safe‑area around the icon ≈ 1/4 of icon width.
- Minimum sizes: icon ≥ 24 px, horizontal lockup ≥ 320 px width.

## Notes
- Vector PDFs were auto‑generated from SVG (CairoSVG).
